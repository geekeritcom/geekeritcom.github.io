HTML简介

HTML（Hyper Text Markup Language）

超文本标记语言

浏览器内核（渲染引擎），读取网页内容，整理信息，计算网页显示方式

web标准

优点

  • 统一页面展示
  • 更易于被搜索引擎搜索
  • 降低网站流量费用
  • 提高页面浏览速度

构成

  • 结构　　对网页元素整理与分类（HTML）
  • 表现　　设置网页元素的版式颜色大小等外观(CSS)
  • 行为　　网页模型的定义与交互(JavaScript)

最佳体验

结构、行为、样式分离

HTML标签

语法规范

  • 所有标签由<>包围
  • 所有标签大部分情况下应该成对出现
  • 特殊标签必须是单标签，例如<br />
  • 后缀名必须是.html或者.htm

文档类型声明标签

\<!DOCTYPE html>

  • 作用是文档类型声明，告诉浏览器使用哪种HTML版本进行显示
  • 必须位于文档最前面进行声明并且该标签属于文档类型声明标签而非HTML标签

图像标签

属性 属性值 说明
src 图片路径 必须属性
alt 文本 图片无法显示时提示文字
title 文本 鼠标置于图片上时显示文本
width 像素 图像宽度
height 像素 图像高度
border 像素 图像边框

注意点

  • 图像标签可以拥有多个属性，必须放在标签名的后面
  • 属性之间不分先后顺序，标签名与属性、属性与属性之间以空格分开
  • 属性采取键值对格式
